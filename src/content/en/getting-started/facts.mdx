# Facts

## Facts

Facts are the smallest learning units used to build cards and decks in Retentio. A fact can be understood as a complete piece of learning content, usually representing a word, concept, knowledge point, or a group of related information that the user needs to learn and remember.

A fact is not necessarily the same as a card. Instead, it is the underlying data used to generate cards. Cards are shown to users during learning and review, while facts store the original content needed to create those cards.

For example, in language learning, a fact can be:

`English: hello`  
`中文: 你好`

This fact represents a complete learning item: the relationship between the English word “hello” and its Chinese meaning “你好”.

## Field Content

A fact usually contains two or more fields and their corresponding content. Fields define what types of information a fact can include, while field content is the actual learning material that users need to study and remember.

For example, when a deck has the fields “English” and “中文”, a user can create a fact such as:

`English: hello`  
`中文: 你好`

In other learning scenarios, a fact can also include more fields, such as term, definition, example, note, image, audio, or pronunciation. These fields help users break down a knowledge point into clearer and more structured information.

Well-structured field content helps the system generate more accurate cards. It also makes it easier for users to understand the relationships between different parts of the knowledge during learning and review.

## Generating Cards

Facts are the basis for generating cards. In most cases, one fact generates one card. The card usually contains a front side and a back side, which are used for memory training.

For example, if a fact is:

`English: hello`  
`中文: 你好`

The system can generate a card like this:

Front: `hello`  
Back: `你好`

In some cases, one fact can also generate multiple cards. For the same fact, the system can generate both an English-to-Chinese card and a Chinese-to-English card:

Front: `hello`  
Back: `你好`

Or:

Front: `你好`  
Back: `hello`

This allows users to practice recall in different directions. They can learn not only to see the English word and remember the Chinese meaning, but also to see the Chinese meaning and recall the English word. This helps improve the completeness of memory and understanding.

## Summary

Facts are the most basic knowledge records in Retentio. They store each piece of learning content and organize different dimensions of information through fields.

By creating clear and complete facts, users can help the system generate more effective cards and further organize them into structured decks. In other words, decks are used to manage learning topics, cards are used for the learning process, and facts are the core foundation behind the learning content.
