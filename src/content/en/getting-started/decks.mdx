# Decks

## Decks

Decks are one of the fundamental units used to organize learning content in Retentio. You can think of a deck as a learning container used to group together all the content related to a specific topic, domain, or learning goal. In simpler terms, it can also be understood as a box that contains many cards.

In most cases, each deck should focus on a single area of knowledge. For example, it may be used to organize vocabulary for a language, key concepts from a subject, or a set of professional terms and facts. By grouping related content into the same deck, users can manage their study materials more clearly and gradually build a more structured knowledge system.

## Fields

When creating a deck, users also need to define the fields used in that deck. Fields represent the different pieces of information contained in a learning item and form the basic structure for how content is displayed, edited, and studied later.

For example, when learning Chinese and English, the fields in a deck may be set as “Chinese” and “English” to store the content for each language. In other learning scenarios, a deck may also contain multiple fields used to represent different dimensions of information, such as terms, definitions, examples, notes, images, audio, or pronunciation.

Properly defining fields helps users organize learning content more clearly, build a more structured system, and make memorization and review more efficient.

## New Cards

In addition, when creating a deck, users also need to choose how many new cards they want to study each day. This setting determines how many new cards the system introduces into the learning process on a daily basis, and it also affects the pace at which new content is presented.

When the daily number of new cards is set higher, new content will be introduced more quickly and the learning process will move forward at a faster pace. When the number is set lower, the learning rhythm becomes more gradual, which may be more suitable for users who prefer a steadier pace.

## Summary

By structuring decks clearly, defining appropriate fields, and setting a suitable daily number of new cards, users can keep their learning content more consistent and make the overall study and review process more organized, resulting in a more stable and effective memory training experience.
