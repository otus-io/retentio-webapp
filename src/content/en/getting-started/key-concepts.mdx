# Key Concepts

To use Retentio more effectively, we recommend first understanding a few core concepts. Retentio’s content organization, learning flow, and review system are all built around these foundational ideas.

In the following sections, we will introduce decks, cards, facts, fields, as well as images and audio, to help you build a clear overall understanding of how Retentio works. Once you are familiar with these concepts, it will be easier to create content, manage your materials, and study more efficiently while making full use of Retentio’s memory training capabilities.
