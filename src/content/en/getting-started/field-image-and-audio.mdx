# Fields, Image and Audio

## Fields

Fields are the basic structure that needs to be defined when creating a deck. Fields describe the different pieces of information contained in a learning item. They also provide the foundation for creating facts, generating cards, editing content, and displaying learning materials.

A field can be understood as a category of information within a learning item. For example, in a language learning deck, users can create two fields: “English” and “中文”. These fields are used to store the English word and its Chinese meaning.

When users add a fact, they fill in the corresponding content based on these fields. For example:

`English: hello`  
`中文: 你好`

The system can then use this field content to generate cards and display the content on the front or back side of a card.

## Text Fields

In most cases, fields are used to store text content. Text is the most common form of learning information and is suitable for words, meanings, concepts, definitions, explanations, example sentences, and notes.

For example, an English learning deck can include the following fields:

`English`  
`中文`  
`Example Sentence`  
`Note`

The “English” field can be used to store the word, the “中文” field can be used to store the Chinese meaning, the “Example Sentence” field can be used to store an example sentence, and the “Note” field can be used to store additional information.

By setting multiple text fields, users can describe the same knowledge point from different angles. This makes the learning content more complete and also makes it easier to understand and remember during review.

## Image Fields

In addition to text, fields can also be used to add images. Images can help users understand and remember learning content through visual information.

For example, when learning objects, scenes, locations, people, animals, plants, or visual concepts, images can provide more direct support than text alone.

In language learning, users can add an image for a word. For example, when learning “apple”, an image of an apple can help users connect the word, its meaning, and the visual impression together.

Image fields are useful for strengthening memory cues, especially when the learning content is easier to understand with visual support.

## Audio Fields

Audio fields can be used to store pronunciation, reading audio, listening materials, or other sound content. They help users learn and review through listening.

In language learning, audio fields are often used for word pronunciation, sentence reading, or short dialogue clips. For example, when learning the English word “hello”, users can not only see the text, but also listen to its correct pronunciation.

Audio fields help users build a connection between written content and sound. They are especially useful for pronunciation training, listening practice, and language memory.

For learning content that requires sound-based understanding or recognition, audio can provide more direct support than text alone.

## Field Combination

A deck can contain multiple fields based on its learning goal. Different fields can be combined to form a complete fact, which can then be used to generate cards.

For example, an English learning deck can include:

`English`  
`中文`  
`Example Sentence`  
`Image`  
`Audio`

When users create a fact, they can enter the English word, Chinese meaning, example sentence, related image, and pronunciation audio. In this way, one fact can contain not only text information, but also visual and audio support.

These fields make card content richer and help users remember knowledge from multiple directions. Users can understand the content through text, build visual associations through images, and strengthen pronunciation and listening memory through audio.

## Summary

Fields are the basic structure for organizing learning content in Retentio. They define what types of information a fact can contain and affect how cards are generated, displayed, and studied.

Text fields are suitable for words, meanings, definitions, example sentences, and notes. Image fields provide visual support. Audio fields support pronunciation, listening, and sound-based memory.

By setting text, image, and audio fields properly, users can make their learning content more complete, clear, and multi-dimensional, which helps improve the effectiveness of later learning and review.
