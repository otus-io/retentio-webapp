# Hard or Easy

## Hard or Easy

Hard or Easy is a feedback step in the Retentio learning process. After users show the answer, they need to evaluate how difficult or easy the current card feels based on their actual memory performance.

This feedback affects the learning algorithm. Based on the difficulty level selected by the user, the system automatically adjusts how often each card appears, the order in which cards are shown, and the future review schedule. This helps the learning process better match the user’s real memory state.

## Difficulty Slider

After users click Show Answer, a difficulty slider appears at the bottom of the page.

The left side of the slider represents Hard, while the right side represents Easy. Users can move the slider to the position that best matches how they felt when recalling the answer.

If users could barely remember the answer, or if the recall process felt difficult, they can move the slider closer to the Hard side. If users could recall the answer quickly and accurately, they can move the slider closer to the Easy side.

This action tells the system how difficult the current card is for the user. The system cannot read the user’s mind, despite humanity’s impressive habit of expecting software to understand everything automatically.

## Choosing Based on Real Feeling

When selecting the difficulty level, users should base their choice on their actual feeling and memory performance.

If a card looks familiar but the user could not truly recall the answer before showing it, the card should still be considered relatively difficult. If the user could quickly remember the answer after seeing the front side, and the answer was mostly correct, the card can be considered relatively easy.

Honest feedback helps the system better understand how well the user has mastered each card. If users choose randomly, the learning algorithm may not be able to arrange future reviews accurately, which can reduce the overall learning effect.

## Review Action

After selecting the difficulty level, users can click the Review button to submit their feedback for the current card and move on to the next card.

After Review is clicked, the system records the user’s difficulty rating for the current card and uses the algorithm to decide its future review schedule. At the same time, the system displays the next card that needs to be studied or reviewed.

This process repeats during learning: view the front side of the card, try to recall the answer, show the answer, select the difficulty level, and click Review to continue to the next card.

## Impact on the Learning Algorithm

The Hard or Easy selection directly affects the Retentio learning algorithm. The system uses the user’s feedback to automatically adjust the frequency and order of future card appearances.

If a card is marked as difficult, the system will usually schedule it to appear more often, helping the user strengthen their memory. If a card is marked as easy, the system can reduce how often it appears in the near future and give more learning time to content the user has not mastered yet.

In this way, Retentio can make the learning process more personalized. The review schedule of each card is adjusted based on the user’s actual performance, instead of simply repeating cards in a fixed order.

## Summary

Hard or Easy is an important step for collecting learning feedback in Retentio. After showing the answer, users can use the slider at the bottom of the page to select the difficulty level of the current card, and then click Review to move on to the next card.

Users should make this choice based on their real memory experience, because the feedback affects the learning algorithm. The system uses the difficulty rating of each card to adjust how often and in what order the card appears, helping users achieve a more reasonable and effective learning result.
