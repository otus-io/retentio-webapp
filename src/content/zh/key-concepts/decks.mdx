# 卡组（Decks）

## 卡组

卡组（Decks）是 Rete 中用于组织学习内容的基础单位。你可以将一个卡组理解为一个学习容器，用来集中管理某一特定主题、领域或学习目标下的材料。它包含你录入的 **事实**（内容）和用于复习的 **卡片**，以及控制结构与学习节奏的设置。

通常情况下，一个卡组应围绕单一的知识范围建立。例如，它可以用于整理某一门语言的词汇、某一学科的核心概念，或某一类专业术语。将相关内容归入同一卡组，可以更清晰地管理学习材料，并逐步建立结构化的知识体系。

## 字段

创建或编辑卡组时，需要定义 **字段（Fields）**：该卡组的列名（例如 `English`、`中文`、`Example Sentence` 或 `Note`）。字段说明每一列的含义，以及编辑事实、复习卡片时如何显示标签。

字段 **不保存** 学习材料，只保存列名与顺序。重命名或调整字段会改变卡组结构；事实 **条目** 仍按位置对应（条目 `0` 始终是第一列，条目 `1` 是第二列，依此类推）。

卡组可以只有一列，也可以有多列。列名应尽量符合你的学习习惯，便于后续管理事实与卡片。

## 每日新卡

创建卡组时，还需设置每天引入多少张 **新卡片**（在 API 中称为 **rate**）。这控制**全新**卡片进入学习队列的速度——**不是**你每天总共要复习多少张卡。

- **较高** 的数值：新内容进入更快。
- **较低** 的数值：节奏更平缓、更从容。

Rete 会在一天内均匀分布新卡。例如，rate 为 20 时，大约每 72 分钟引入一张新卡；rate 为 10 时，大约每 144 分钟一张。

## 小结

卡组归纳某一主题下的事实与卡片，定义列 **字段**，并设定每日 **新卡** 节奏。清晰的卡组与字段布局，能让后续添加事实、生成卡片和复习更加可预期、更高效。
