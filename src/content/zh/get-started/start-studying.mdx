# 开始学习

卡组中已有事实与卡片后（见[创建卡组](/guide/get-started/create-your-deck)），打开卡组即可复习。Rete 会选出当前最该学习的到期卡片——也可按[标签](/guide/key-concepts/tags)筛选——并引导你完成三步：**回忆**、**显示答案**、**选择难易**。

## 看正面并回忆

先显示卡片 **正面**：来自事实的一条或多条 **条目**（界面常带字段标签，如 `English:`）。

示例：

`English: hello`

请先自行回忆答案，再翻开背面——例如 `こんにちは`。主动回忆比「看着眼熟」更有效。

正面也可包含图片、音频或视频，取决于条目与卡片布局。

## 显示答案

准备好后，点击卡片或页面底部 **显示答案**。

**背面** 出现，用于核对回忆是否正确：

正面：`hello`  
背面：`こんにちは`

记得准说明已有一定基础；记得不准则会在后续复习中更频繁出现。

## 困难或简单

答案显示后，使用页面底部的 **难易滑动条**。**困难** 在左，**简单** 在右——根据真实回忆感受选择。

- 几乎想不起来或很吃力 → 偏 **困难**
- 快速、准确想起 → 偏 **简单**

请如实选择，随意打分会影响调度。

点击 **Review** 保存评价并进入下一张卡。算法据此安排下次出现时间：越难的卡越早再出现，越简单的卡间隔可拉长。

## 学习循环

每张卡重复同一流程：

1. 看 **正面** 并回忆
2. **显示答案** 并核对
3. 选择 **困难或简单**，**Review** 进入下一张

持续这一循环，才是间隔重复训练记忆的方式，而不是被动浏览内容。

## 小结

**开始学习** 是间隔重复发生的地方：正面 → 显示答案 → 难易 → 下一张。内容准备在[创建卡组](/guide/get-started/create-your-deck)；本页说明日常复习流程。
